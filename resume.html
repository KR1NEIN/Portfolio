Private Investment Team 产品设计

项目简介

从第一性原理出发，针对投资者常见的信息过载、执行变形、缺乏复盘三大痛点，独立设计了一套面向美股市场的 Multi-Agent 数字投研团队。系统包含 Portfolio Manager、Macro Analyst、Technical Strategist、Risk Manager、Backtester 五个模块，模拟对冲基金中台的运作流程，实现情报采集→策略生成→风控拦截→交易执行→归因进化的全链路自动化。

Agent 架构设计与产品定义

拆解投行投研流程，将业务抽象为 5 个差异化 Agent 角色，构建了具备自我进化能力的反馈闭环。定义各 Agent 的 System Prompt、输入输出 Schema（JSON）及交互协议，明确 Portfolio Manager 作为唯一 Orchestrator 的调度逻辑，解决了 Multi-Agent 协作中的意图冲突与信息冗余问题。

模型选型与成本效能优化

基于业务特性，制定了差异化模型选型策略：风控/复盘采用低成本、高稳定模型；PM 使用多模态旗舰模型；基本面和技术面分析采用强推理、长上下文模型。在不牺牲核心能力的前提下，通过模型分层来降低成本。

评测体系搭建

设计了执行缺口评测指标与 Signal-to-Noise Ratio 监控机制，通过复盘 Agent 持续追踪各技术形态的胜率，反向指导前端策略参数的迭代优化。针对幻觉问题，通过在 System Prompt 中添加硬约束规则，如风控 REJECT 不可被覆盖，提升系统稳定性。

用户体验优化

针对盘中无暇打字的问题，设计了 Trade Log 三种模式输入，包括完整模板、极简文本和截图识别，压缩收盘后的记录时间。将投研数据封装为标准化的 IC Memo（投资委员会备忘录），仅向用户显示关键决策的信息，例如方向、出入场点、仓位以及置信度，降低用户接收信息的认知负荷。
