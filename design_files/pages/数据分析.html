数据分析

围绕"数据飞轮"核心假设，定义度量指标、验证节奏与决策规则

Core Hypothesis

免费搭配工具吸引用户生产搭配数据 → 协同过滤引擎消费数据生成推荐方案 → 推荐方案带动更多购买 → 更多用户贡献数据。度量体系围绕飞轮是否转起来设计，而非单纯追踪 GMV。

双路径转化漏斗

产品存在两条核心用户路径，需分别度量转化效率与数据产出

PATH A 消费推荐数据

目标：提升搭配采用率与客单价

看到推荐方案 基准 100%
点击查看详情 目标 ≥ 60%
收藏 / 加入购物车 目标 ≥ 30%
完成购买 目标 ≥ 12%
关键流失监控：推荐 → 详情点击（方案吸引力） · 购物车 → 支付（价格阻力）
PATH B 自由搭配 · 生产数据

目标：最大化有效搭配数据产出

进入自由搭配 基准 100%
添加 ≥ 2 件饰品 目标 ≥ 55%
保存搭配方案 目标 ≥ 25%
数据入引擎（≥ 3 件 · 跨品类） 目标 ≥ 15%
关键流失监控：添加入口曝光 → 进入（入口可见性） · 保存 → 数据入引擎（方案质量门槛）

核心指标定义

每个指标明确计算公式、数据来源、V1 目标值及对应决策规则

指标 计算公式 数据源 V1 目标 决策规则
搭配采用率 被收藏或加购的方案数 ÷ 推荐方案曝光数 推荐引擎 ≥ 20% < 15% → 检查推荐算法权重与种子方案质量；15-25% → 优化展示形式；> 25% → 扩大推荐覆盖范围
推荐购买转化率 通过推荐路径完成的订单数 ÷ 推荐方案曝光数 交易系统 ≥ 8% < 5% → 排查价格阻力与购物车体验；5-8% → 优化方案内价格区间匹配；> 8% → 验证飞轮商业可行性
搭配客单价 推荐路径订单总金额 ÷ 推荐路径订单数 交易系统 高于均值 30% 低于非推荐订单 → 推荐方案偏向低价品，需调整价格带匹配策略
自由搭配使用率 进入自由搭配的用户数 ÷ 功能入口曝光用户数 行为埋点 ≥ 40% < 25% → 入口设计或引导不足，优化曝光位置与 CTA；> 40% → 数据飞轮供给充足
有效数据产出率 符合入引擎条件的方案数 ÷ 总搭配方案数 搭配工具 ≥ 60% < 40% → 降低入引擎门槛或引导用户添加更多跨品类饰品；> 60% → 数据质量健康
数据驱动覆盖率 协同过滤生成方案数 ÷ 总推荐方案数 推荐引擎 V1 末 ≥ 50% < 30% → 种子方案占比过高，需加大搭配工具推广以积累数据；> 50% → 飞轮初步转动
跨用户被采用率 被其他用户采纳的 UGC 方案数 ÷ 总 UGC 方案数 推荐引擎 ≥ 10% < 5% → UGC 方案质量参差，需增加筛选或排序机制；> 10% → 社区共创价值验证

数据飞轮健康度

监控飞轮三个关键环节的运转状态，任一环节停滞需立即干预

SUPPLY · 数据供给
自由搭配数据产出
周搭配方案产出量 目标 ≥ 5,000
人均搭配件数 目标 ≥ 3.5 件
跨品类搭配占比 目标 ≥ 70%
低于目标 → 优化搭配工具引导、增加入口曝光、提供搭配模板
PROCESS · 引擎转化
推荐引擎消费数据
数据驱动方案占比 目标 ≥ 50%
推荐覆盖率 目标 ≥ 80%
方案平均热度分 目标 ≥ 60
覆盖率低 → 数据量不足以支撑协同过滤，回退种子方案补充
OUTCOME · 价值产出
推荐驱动的业务结果
推荐路径 GMV 占比 目标 ≥ 15%
搭配复访率（7 日） 目标 ≥ 25%
UGC 跨用户采用率 目标 ≥ 10%
复访率低 → 推荐结果缺乏新鲜感，需引入趋势与季节因子
飞轮运转逻辑
用户自由搭配 → 搭配数据入库 → 协同过滤生成推荐 → 用户采纳购买 → 更多用户参与搭配 ↻

V1 实验计划

V1 阶段聚焦验证飞轮假设，实验设计围绕"推荐是否有效"和"数据驱动是否优于种子"

实验一：协同过滤 vs 种子方案

待启动
对照组 A
种子方案（运营编辑）
基线
实验组 B
协同过滤推荐
待测量
主指标搭配采用率
辅指标购买转化率、客单价
启动条件搭配数据 ≥ 10,000 条
实验周期2 周（含周末）
显著性要求p < 0.05
决策：B 显著优于 A → 全量切换协同过滤；无显著差异 → 优化算法权重后重测

实验二：推荐展示 vs 无推荐

待启动
对照组 A
仅展示饰品列表
基线
实验组 B
展示搭配推荐模块
待测量
主指标平均客单价
辅指标跨品类购买率、人均购买件数
启动条件推荐功能上线即启动
实验周期3 周
显著性要求p < 0.05
决策：B 客单价显著高于 A → 验证推荐带动关联购买；无差异 → 推荐模块价值需重新评估

度量节奏

不同阶段关注不同指标集合，避免过早优化次要指标

WEEK 1-2
数据采集验证
  • ✓ 埋点完整性校验
  • ✓ 自由搭配使用率
  • ✓ 搭配方案产出量
  • — 暂不关注转化率
WEEK 3-4
飞轮启动验证
  • ✓ 搭配采用率
  • ✓ 有效数据产出率
  • ✓ 数据驱动覆盖率
  • ✓ 实验二启动（推荐 vs 无推荐）
WEEK 5+
算法效果验证
  • ✓ 实验一启动（CF vs 种子）
  • ✓ 推荐购买转化率
  • ✓ 搭配客单价
  • ✓ 跨用户被采用率
